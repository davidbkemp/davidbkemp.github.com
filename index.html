Here are some projects I am working on:

  • Quantum Gate Playground: A web application intended to help quantum computation students explore and play with a number of different quantum computer gates. It is built using Animated Qubits.
  • Animated Qubits: JavaScript library for animating quantum gate based quantum computations. It is built on top of jsqubits.
  • jsqubits: A JavaScript library for simulating quantum computation (gate based). It just does simulation, it does no graphical animation.
